# PinTop

> 按下快捷键，一键置顶任意 Windows 窗口

## 简介

PinTop 是一个轻量级窗口置顶工具。按 `Ctrl+Alt+↑` 即可将当前活动窗口置顶或取消置顶。置顶窗口顶部中间会自动吸附一个红色 📌 图标，你扫一眼就知道哪些窗口被钉住了。支持同时置顶多个窗口，每窗口有独立图标。快捷键随时可改。

## 运行环境

本脚本需要 AutoHotkey v2 运行环境。推荐用[不忙脚本盒子](https://www.bm-box.cn)运行，盒子自动配置环境依赖，提供开机自启、进程保活、快捷键管理等功能，开箱即用。

## 安装方法

1. 打开**不忙脚本盒子**
2. 进入「脚本市场」
3. 搜索「**PinTop**」
4. 点击「安装」

注：在不忙脚本盒子任务设置中未pintop设置倒计时0秒执行、启动盒子后会自动运行pintop脚本、后面置顶窗口直接按脚本快捷键即可

## 使用方法

### 快捷键

| 快捷键              | 功能           |
| ---------------- | ------------ |
| `Ctrl + Alt + ↑` | 切换当前活动窗口置顶状态 |

**自定义快捷键：** 右键托盘图标 → 编辑快捷键 → 按新组合键 → 保存，即时生效。

**恢复默认：** 右键托盘图标 → 恢复默认快捷键。

### 托盘菜单

| 菜单项     | 说明             |
| ------- | -------------- |
| 编辑快捷键   | 打开快捷键设置窗口      |
| 恢复默认快捷键 | 恢复为 Ctrl+Alt+↑ |
| 重新加载    | 重新加载脚本         |
| 暂停热键    | 临时禁用快捷键        |
| 暂停脚本    | 暂停脚本运行         |
| 退出      | 退出脚本并取消所有窗口置顶  |

## 输出结果

- 置顶/取消时，鼠标旁显示提示
- 置顶窗口顶部显示 📌 图标，随窗口移动、最小化隐藏、关闭销毁
- 脚本退出时自动取消所有窗口置顶
